<h2 id="pre-requisitos">Pre-requisitos</h2>

* Node (v16^)
* Git
* Yarn

<h2 id="instalar-projeto">Como instalar o projeto</h2>
<h3>
Baixar Projeto 
</h3>

```
git clone https://github.com/JavelFreitas/reactjs-refactoring-classes-ts-ignite.git
```
<h3>
Instalar as Dependências 
</h3>

```
yarn
```
<h3>
Criar uma build 
</h3>

```
yarn build
```
<h3>
Rodar 
</h3>

<h4>
Terminal 1
</h4>

```
yarn server
```
<h4>
Terminal 2
</h4>

```
yarn start
```

<h3>
Testar (Opcional) 
</h3>

```
yarn test
```
<h2 id="funcionalidades">Funcionalidades</h2>

  - Título do prato
  - Preço em Real
  - Descrição do prato

* Editar prato
* Tornar prato "disponível" ou "Indisponível"
